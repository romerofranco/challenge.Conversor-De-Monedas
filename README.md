## Conversor de Monedas 💱

Proyecto desarrollado en Java que permite convertir valores entre diferentes monedas utilizando una API llamada ExchangeRate de tasas de cambio en tiempo real.

### 📌 Descripción

Este programa solicita al usuario un monto y una moneda de origen, luego consulta una API de conversión de divisas para calcular el valor equivalente en otra moneda.

El objetivo del proyecto es practicar:

- Consumo de APIs
- Manejo de JSON.
- Programación en Java
- Uso de Git y GitHub

### Tecnologías utilizadas

- Java

- IntelliJ IDEA

- Git

- GitHub

- API ExChangeRate
